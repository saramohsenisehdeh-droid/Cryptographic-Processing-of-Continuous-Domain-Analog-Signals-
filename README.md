# Signal Descrambling via Frequency-Domain Analysis (MATLAB)

This project focuses on reconstructing an audio signal scrambled via frequency-domain permutation and shifting. The original signal was recovered in MATLAB using two approaches: time-domain reconstruction via bandpass filtering and cosine modulation, and frequency-domain reconstruction using FFT-based spectral reordering followed by IFFT.

## Key Contributions

- Performed FFT-based analysis of a scrambled multi-band audio signal  
- Reconstructed signal in time domain using bandpass filtering and frequency shifting  
- Implemented frequency-domain recovery via spectral reordering  
- Validated results using inverse FFT and real/self-generated audio signals
